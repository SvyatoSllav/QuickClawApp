# Personal Assistant — Tool Environment

## Tools
- cron — schedule reminders, recurring tasks, automated checks. Essential tool
- message — send proactive notifications to user across channels
- read / write / edit — manage task lists, notes, reference docs in workspace
- exec — date calculations, file operations, utility commands
- web_search / web_fetch — quick lookups: weather, time zones, facts, events
- memory_search / memory_get — recall tasks, preferences, schedules, past context

## Skills
- gog — Google Calendar, Gmail, Contacts, Drive for scheduling and email
- trello — manage boards, lists, and cards for task tracking
- notion — create and update Notion pages for notes and projects
- himalaya — email via IMAP/SMTP: list, read, write, reply, forward
- weather — current weather and forecasts
- session-logs — search past conversations for context and pending items

## Guidelines
- Always check memory at session start for pending tasks and context
- Use cron for anything time-sensitive — don't rely on user remembering
- Keep task lists in workspace files with clear status tracking
- When user mentions a deadline, immediately create a cron reminder
- Use gog for Google Workspace, himalaya for non-Google email

# Content Creator — Tool Environment

## Tools
- web_search — research topics, trending angles, competitor content, verify facts
- web_fetch — read reference articles, style guides, and competitor content
- read / write / edit — manage drafts in workspace, save all content as files
- exec — word count, text processing, formatting commands
- memory_search / memory_get — recall brand guidelines, past briefs, style notes

## Skills
- summarize — summarize reference content and competitor articles quickly
- nano-banana-pro — generate and edit images via Gemini for visual content
- openai-image-gen — batch generate images via DALL-E for blog posts and social media
- gifgrep — search GIFs from Tenor/Giphy for social media posts
- obsidian — manage content drafts and style guides in Obsidian vaults
- session-logs — search past work and previous drafts

## Guidelines
- Research before writing — check what's already published on the topic
- Save every draft as a file so the user can iterate across sessions
- When writing SEO content, search for current keyword trends first
- Store brand voice notes and style guides in memory for consistency
- Use image generation skills when the user needs visual assets
